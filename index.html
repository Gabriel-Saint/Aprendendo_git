Resumo de git e terminal(linux)

Comandos Git

  1. git init - inicia o git
  2. git status - verificar o status
  3. git add + "nome_do_arquivo" - adiciona o aquivo específico
  4. git add . - adiciona todos arquivos
  5. git commit -a(para todos tipo de aquivo) -m "mensagem_para_nortear" - faz o commit
  6. git checkout - b + "nome_do_branch" - cria um novo branch
  7. git checkout + "nome_do_branch" - vai para um branch já existente
  8. git switch -c + "nome_do_branch" - cria um branch e já vai ele
  9. git clone + URL +(opcional) "nome_do_projeto" - faz a clonagem
  10. git merge + "nome_do_branch" - faz a integração das alterações
  11. git remote -v - lista os repositórios associados
  12. git remote add origin + URL - Adiciona um repositório remoto
  13. git push origin main - Envia as alterações para o branch main
  14. git remote remove origin - remove repositório remoto
  15. git pull origin main - busca o aquivo e deixa como está no remoto

Comandos terminal(linux)

  1. mkdir + "nome_da_pasta" - cria uma nova pasta
  2. cd + "nome_da_pasta" - entra na pasta
  3. cd .. - volta para o diretório pai
  4. rm -r + "nome_da_pasta" - apaga a pasta
  5. ls - listar as pastas
  6. pwd - ver em qual diretório está
